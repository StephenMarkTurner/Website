Multi Timer

Timer Information

There is an optional delay, to give you time to get ready if needed.

There is one main timer (or it can run without stopping).

You can add one or more 'split' timers (single or repeating). For example, I like to hear a beep every 5 minutes or so when doing Cardio.

You can add one or more 'double' timers, which allow you to set 2 different intervals which alternate. For example, I like a 40 second on / 20 second rest timing for Plank exercises.
