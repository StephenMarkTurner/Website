Super Duper Fitness Timer

Timer Information

Count down to stop after a set period of time.
Count up to stop whenever you finally decide that you are finished.

You can set an initial delay, to move from the computer to your desired location.
You can run a single time, or have intervals.
You can have two intervals, to allow a different work and rest time for example.
You can have a 'heads up' just before each work interval.
You can pause and restart the timer.
